# 🏠 Projetos Caseiros

> Repositório pessoal de projetos desenvolvidos em casa para uso e consulta em sala de aula.

---

## 📌 Sobre este repositório

Este repositório serve como um **portfólio pessoal de estudos**, reunindo projetos criados no tempo livre para que eu possa acessá-los facilmente durante as aulas. A ideia é ter tudo organizado e versionado em um só lugar, sem depender de pen drives ou arquivos locais.

---

## 🗂️ Estrutura

```
/
├── projeto-1/
├── projeto-2/
└── ...
```

> Cada pasta contém um projeto independente com seus próprios arquivos e, quando necessário, um README interno explicando como rodá-lo.

---

## ⚠️ Observação

Este repositório é de **uso pessoal**. Os projetos aqui não seguem necessariamente boas práticas de produção — são experimentos, exercícios e rascunhos feitos para aprendizado.
